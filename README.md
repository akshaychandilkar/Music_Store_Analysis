# Music_Store_Analysis

Music Store Analysis using SQL 

- Developed SQL queries to analyze online music store data using PostgreSQL and PgAdmin4. 
- Extracted insights on customer behavior, sales trends, and playlist performance to support business growth.
- Optimized data retrieval and reporting for key metrics, enabling data-driven decision-making for store 
  management.
